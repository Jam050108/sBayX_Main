# 深贝思 sBayX - 中国企业北美出海基地

## 项目简介

深贝思sBayX是一个专业的一站式服务平台，致力于帮助中国企业在北美市场成功落地、合规运营、持续发展。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 打开网页
   - Local:        http://localhost:3000
   - Network:      http://0.0.0.0:3000
```
